# Python Fighters - Arcade Edition

Um jogo de luta arcade clássico com 8 personagens únicos!

## Modos de Jogo
- **Survival:** Enfrenta inimigos infinitos
- **Local 1v1:** Duel com outro jogador
- **CPU:** Combate contra a IA
- **Online:** Multiplayer via Firebase
- **Tutorial:** Aprende as mecânicas

## Recursos
- 8 lutadores únicos com habilidades especiais
- Loja de personagens desbloqueáveis
- Sistema de cofre mistério para prêmios
- Melhor de 3 rodadas
- Estética CRT arcade clássica
- Suporte mobile completo

Escolha seu lutador e entre na arena! 🥊
