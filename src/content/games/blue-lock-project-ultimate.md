# Blue Lock: Project Ultimate

O treino extremo de futebol temático Blue Lock! Escolhe um dos 9 egoístas e torna-te a melhor máquina de golo.

## 9 Personagens (Egoístas)
Isagi, Bachira, Rin, Sae, Kunigami, Reo, Barou, Shidou, Nagi

## Sistema de Jogo
- **Shooting Mechanic:** Remata bolas contra obstáculos para marcar golos
- **Ego System:** Acumula EGO para ativar "ZONA" (modo flow)
- **Ranking:** 8 níveis (D → C → B → A → S → SS → SSS → EGOIST)
- **Obstáculos:** Drones, goleiros, muros, lasers, plataformas

## Recursos
- Sistema de combo aéreo com multiplicador
- Collectíveis: Bolas ⚽, Visão Meta, Peças de Puzzle
- Progressão dinâmica de dificuldade
- Efeitos visuais e shake dinâmicos
- Suporte mobile completo
- SaveGame com LocalStorage

Devoura os obstáculos e torna-te um egoísta supremo! ⚽
