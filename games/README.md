# Games Collection - From Rafael & Goncalo

Four fully-featured web games added to the workspace.

## 🎮 Games Included

### 1. **Python Fighters - Arcade Edition**
- **File:** `python-fighters.html`
- **Type:** Fighting game (arcade-style, 2.5D fighters)
- **Features:**
  - Multiple game modes: Survival, Local 1v1, CPU battle, Online (Firebase), Tutorial
  - 8 unique fighters with different mechanics (Toxic, Amethyst, Inferno, Abyss, Shadow, Aurum, Cyborg, Carnage)
  - Shop system with coin economy
  - Mystery chest loot system
  - Best-of-3 round system
  - Touch & keyboard controls
  - Online multiplayer via Firebase Realtime Database
  - Special abilities and combos
  - Full CRT arcade aesthetic

**Controls:**
- Keyboard: WASD for movement, F/Space for attack, G for special
- Touch: On-screen D-pad + action buttons
- Online: Firebase integration (requires config)

---

### 2. **Treino Plus Ultra - Ultimate**
- **File:** `treino-plus-ultra.html`
- **Type:** Action/Bullet-hell survival game (My Hero Academia themed)
- **Features:**
  - 9+ hero characters with unique stats and abilities
  - 3 game modes: Combat (survive waves), Rescue (save falling civvies), Boss fight
  - 3 difficulty levels for boss mode
  - Ultimate ability system (100% meter activation)
  - Combo counter with multiplier system
  - Progressive difficulty scaling
  - Collectible power-ups: shields, speed boost, fire rate boost, healing
  - Hero unlock system (tied to score milestones)
  - LocalStorage save system for progression
  - My Hero Academia soundtrack/theme aesthetic
  - Full responsive design with touch controls

**Controls:**
- Keyboard: WASD / Arrow keys for movement
- Touch: Tap to move toward cursor
- Space: Activate Ultimate ability (when meter full)
- Mouse/Touch aim: Move player toward cursor position

**Heroes:** Deku, Uraraka, Froppy, Bakugo, Iida, Chargebolt, Tsukuyomi, Todoroki, Kirishima, All Might

---

### 3. **Blue Lock: Project Ultimate**
- **File:** `blue-lock-project-ultimate.html`
- **Type:** Football/Soccer shooting game (Blue Lock anime theme)
- **Features:**
  - 9 playable characters (Isagi, Bachira, Rin, Sae, Kunigami, Reo, Barou, Shidou, Nagi)
  - Shooting/aiming mechanics with precision hitting system
  - Ego system that builds toward "Zone" flow state
  - Rank progression (D → C → B → A → S → SS → SSS → EGOIST)
  - Special abilities & character-specific mechanics
  - Multiple obstacle types (drone, goalkeeper, walls, lasers, platforms)
  - Combo system with air combos
  - Power-ups: ball ammo, vision enhancement, puzzle pieces
  - Progressive difficulty scaling
  - LocalStorage save system

**Controls:**
- Keyboard: WASD for movement, X/Shift for shoot, Space/W for jump, S for slide
- Touch: On-screen buttons
- Goal: Shoot balls at obstacles to score points and build ego

---

### 4. **Blue Lock: Parkour Egoísta - Ultimate**
- **File:** `blue-lock-parkour-egoista.html`
- **Type:** Parkour/Platformer action game (Blue Lock anime theme)
- **Features:**
  - 9 playable ego characters with unique abilities
  - Parkour platforming: jumping, sliding, double-jump, triple-jump (with boost)
  - Obstacle dodging with "rasante" (close-call) bonus mechanics
  - Collectibles: Ego pieces, health shields, speed boosts, jump boosts
  - Combo system based on stomping obstacles
  - Screen shake & visual feedback effects
  - Progressive game speed increase
  - Holographic enemy aesthetic
  - Character-specific stat variations
  - Mobile responsive with touch controls

**Controls:**
- Keyboard: Space/W for jump (2 jumps default), S for slide, X/Shift for shoot
- Touch: On-screen D-pad style controls + action buttons
- Gameplay: Navigate obstacles, build combo multiplier, unlock zone mode via ego

## 🚀 How to Play

1. **Python Fighters:**
   - Open `python-fighters.html` in a modern browser
   - Select game mode (CPU, Local 1v1, Online, Survival, etc.)
   - Pick a fighter
   - Use keyboard (WASD + F/G) or touch controls

2. **Treino Plus Ultra:**
   - Open `treino-plus-ultra.html` in a modern browser
   - Pick a mission type (Combat/Rescue/Boss)
   - Select difficulty (for boss)
   - Choose a hero
   - Survive & accumulate points
   - Use SPACE to activate Ultimate when bar fills

## 📝 Notes

- Both games use **HTML5 Canvas** for rendering
- **Python Fighters** includes Firebase integration (online multiplayer requires Firebase config)
- **Treino Plus Ultra** uses LocalStorage for save data (persistent progression)
- Both are fully self-contained (single HTML files, no external dependencies beyond CDNs)
- Mobile-responsive with touch controls
- Estimated play session: 5-30 minutes per game
